# Proyek Analisis Data

## Set Up Environtment
```
conda create --name main-ds python=3.11
conda activate main-ds
pip install numpy pandas matplotlib seaborn jupyter streamlit babel
```

## Run steamlit app
```
streamlit run dashboard.py
```
